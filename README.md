- Projeto pessoal simples de controle financeiro feito em Java com JSP, Servlet, MySQL e Tomcat.
 

- Funcionalidades 

CRUD de receitas e despesas

Visualizar o saldo (orçamento) e o total de receitas e despesas


- Tecnologias usadas

Java

Servlets e JSP

MySQL

Tomcat

HTML, CSS, JavaScript
